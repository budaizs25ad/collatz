🔢 A Collatz-sejtés

Egy híres, máig megoldatlan matematikai probléma. A szabály egyszerű: bármely pozitív egész szám esetén a sorozat végül a 4 → 2 → 1 ciklusba jut.

an+1 = { páros → an / 2; páratlan → 3an + 1 }

ahol n pozitív egész szám

n

Az értékek itt lesznek kilistázva.
